# UltraEdit Clone

A modern, fully responsive **UltraEdit landing page clone** built using HTML, CSS, and JavaScript.  
This project focuses on responsive design, layout systems (Flexbox & Grid), and interactive UI elements like sliders.

---

## 🚀 Live Demo
👉 https://maazarifdev.github.io/ultraedit-clone/

---

## 📌 Project Overview

This project is a frontend clone of the UltraEdit website homepage.  
It replicates the original design with a focus on:

- Clean UI structure
- Responsive layouts for all devices
- Modern CSS techniques
- Smooth user experience

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Swiper.js (for slider functionality)
- Flexbox & CSS Grid
- Media Queries (Responsive Design)

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive image slider using Swiper.js
- Modern hero section layout
- Responsive navigation bar
- Grid-based feature section
- Flexible card layouts
- Smooth hover animations
- Organized folder structure
